## Usage 🚀

1. Clone the repository:

```bash
git clone https://github.com/Axlecreel/discord-selfbot.git
cd discord-selfbot
```

2. Create and activate virtual environment:

```bash
sudo apt update && sudo apt install python3-venv
python3 -m venv venv #can change the last venv to any be sure to change the activation
source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
```

3. Install dependencies:

```bash
pip install -r requirements.txt  # Or use `uv pip install` if available
```

4. Run the bot:

```bash
python3 bot.py
```
